```

```

## フェーズ1の実装ゴール

React + Hono + Cloudflare Workers + D1 を繋いだメモアプリを1本完成させる。

### API仕様

| メソッド | エンドポイント | 処理 |
|---|---|---|
| GET | /api/memos | 一覧取得 |
| POST | /api/memos | 新規作成 |
| DELETE | /api/memos/:id | 削除 |

### データ構造

```ts
{
  id: number,
  content: string,
  created_at: string
}
```

### 実装ステップ

1. 配列をデータストアとして3つのエンドポイントを実装する（DBなし）
2. Cloudflare Workers でローカル動作確認
3. D1にテーブルを作ってAPIと繋ぎ替える
4. Reactフロントと接続して完成

## 指摘の方針

コードレビュー時は以下を優先する：

1. **動く理由・動かない理由**を説明させる
2. `return` 抜け・パスの `/` 抜けなどのケアレスミスは指摘する
3. 過剰なリファクタや先回りした設計提案はしない（今必要なことだけ）
4. 詰まったときに「これは本で体系的に学ぶべきか、今は記事で十分か」を判断してガイドする
